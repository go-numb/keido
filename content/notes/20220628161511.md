### markmap {#614a1d02-bfc2-488d-9388-ba2a2203f450}

[markmap](https://markmap.js.org/) という [Markdown]({{< relref "20211211171552.md" >}}) から マインドマップへの変換ツール

```sh
# install
$ npm install -g markmap-cli

# export
$ markmap note.md
```
